## Một vài tùy chỉnh quan trọng!
Trước tiên chạy 3 lệnh này:
```
/lp import "luckperms-2025-02-16-20-43.json.gz"
/gamerule playersSleepingPercentage 50
/rs install Terralith
```
Sau đó restart và chạy tiếp 2 lệnh này:
```
/rs set spring world
/timebar reload
```
Xong!
