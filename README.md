## Some important settings!
First, run these commands:
```
/lp import "luckperms-2025-02-16-20-43.json.gz"
```
```
/gamerule playersSleepingPercentage 50
```
```
/rs install Terralith
```
Then restart and run these commands:
```
/rs set spring world
```
```
/timebar reload
```
Done!
